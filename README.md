## 鼠标输入法

基于Qt制作的一款鼠标输入法。



#### 使用方法

运行安装程序，根据指引进行操作。



#### 授权协议

程序代码部分遵循[WTFPL]( http://www.wtfpl.net/about/ )发布。请任意地使用吧。

###### 注意：

图形库[Qt](www.qt.io)，词库[THUOCL](http://thuocl.thunlp.org/ )(已从当前版本中移除)，词库[SogouW](https://www.sogou.com/labs/resource/w.php) 分别遵循各自的协议发布。
如果需要使用到它们，请遵循其相关协议。
